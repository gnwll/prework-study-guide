HTML

  • HTML - Hypertext Markup Language
  • HTML is the basic framework for all pages on the web.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The base element defines the base URL for a webpage.
  • Text elements, image elements, header elements, list elements, are all common elements used to add content.
  • Meta-related elements, content division elements, semantic elements, are used to divide content.

CSS

  • A margin indicates how much space is around the outside of an element.
  • A padding indicates how much space is around the content inside an element.
  • Inline styles apply to specific HTML tags. And are less flexible compared to external and internal styles.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • External CSS style sheet allows developers to keep their CSS rules in a separate file, making design changes easier.
  • A selector defines the element or attributes.
  • Instead of repeating a rule, two or more elements that share the same styling can be grouped together.
  • card is a class selector, which a CSS rule can be assigned to.

Git

  • Git version control is used to keep track of content changes.
  • Git command must precede every Git statement.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: receive branch's modifications into local computer.
  • Push changes with, `git push origin feature/example`, pushes changes from local computer to remote GitHub branch.

JavaScript

  • JavaScript is an object-oriented programming (OOP) language.
  • JavaScript a high-level dynamic programming language, allowing users to interact with websites.
  • In JavaScript the order of code affects the outcome, because it reads from top to bottom.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Commenting out code makes it inactive without deleting.
  • Conditional logic is a simple way to tell a computer to perform an action when a certain condition is met.
  • An array is a single variable that is used to hold a group of data. This data is typically related in some way.
  • `If` statements are used to introduce decision-making into code.
